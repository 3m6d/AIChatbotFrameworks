# AIChatbotFrameworks

Learning and experimenting with different chatbot frameworks.

## Overview

This repository contains my journey of learning and experimenting with various chatbot frameworks. Specifically, I have explored the Microsoft Bot Framework and Rasa Framework.

### Microsoft Bot Framework

In this section, I have used the Bot Builder SDK to create a simple financial bot that answers queries related to financial capabilities.

#### Projects:
- **Finbot**: An intelligent bot that provides financial information about Nepal using the [Bot Framework](https://dev.botframework.com).
- **Echo Bot**: A simple bot that echoes user inputs back to them.

### Rasa Framework

In this section, I have learned to build chatbots using the Rasa Framework and developed a simple financial bot.
